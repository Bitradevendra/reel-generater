TROUBLESHOOTING
====================================================================

Q: Script hangs or is very slow?
A: Model loading can take 1-3 minutes on first run
   - CPU-only processing is slower
   - Larger model files take longer
   - Try shorter durations (5-10 seconds) first

Q: "Model not found" error?
A: Download from:
   https://huggingface.co/bartowski/deepseek-r1-distill-qwen-7b-gguf
   - Download Q4 version (~4.7 GB)
   - Rename to: model.gguf
   - Place in: c:\Users\papan\Downloads\script\

Q: Out of memory error?
A: Reduce max_tokens in main.py for shorter outputs
   Or close other applications

Q: Poor quality output?
A: - Try different content types
   - Increase duration for better flexibility
   - Try audience adjustment (engineers vs common)
